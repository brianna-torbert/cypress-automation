# Cypress QA Automation Project

## Overview
This project demonstrates end-to-end UI and API test automation using **Cypress**.  
It is structured to reflect real-world QA practices, including organized test suites, reusable commands, and scalable folder design.

The goal of this repository is to showcase practical QA automation skills applicable to modern web applications.
